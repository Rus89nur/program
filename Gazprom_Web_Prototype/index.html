💾 Резервные копии

Экспорт — скачать копию

Сохраните все текущие данные браузера в файл .json. Файл можно открыть в любом браузере, в том числе на телефоне.

Импорт — загрузить копию

Файлы резервной копии (.json или .gazprombackup) можно загрузить здесь.

📱 Файл .gazprombackup не выбирается на телефоне?
  1. Откройте приложение Файлы на iPhone
  2. Найдите файл .gazprombackup
  3. Удерживайте палец на файле → нажмите Переименовать
  4. Измените расширение: .gazprombackup → .json
  5. Вернитесь сюда и выберите переименованный файл
