Резервная копия с iPhone

Файлы .gazprombackup создаются в iOS-приложении автоматически. Чтобы перенести на компьютер:

  1. iPhone → Настройки → Восстановление по дате (календарь бэкапов)
  2. Выберите дату → копию → Поделиться
  3. Отправьте себе на Mac (AirDrop, «Файлы», почта, iCloud Drive)
  4. Загрузите файл ниже или перетащите на эту карточку

Скачать демо-файл для проверки (без фото, ~2 КБ).
